[实验题5-4]
题目描述

编程找出这样的三位数,其每一位数字的立方和正好等于这个三位数本身,如371就是这样一个数,它满足33+73+13=371。满足这种条件的数也叫“水仙花数”。
输入
无
输出
从小到大输出所有的水仙花数，以空格分隔
样例输入

无


#include<stdio.h>
#include<math.h>
int main() {
	for(int i=100;i<1000;i++)
	{
		if(pow((i/100),3)+pow((i/10%10),3)+pow((i%10),3)==i)
		{
			printf("%d ",i);
		}
	}
	return 0;
}
以下是对这段代码的解释：
一、整体功能
这段 C 语言代码的作用是找出所有的三位数水仙花数并打印出来。水仙花数是指一个三位数，其各位数字的立方和等于该数本身。
二、具体步骤
外层循环遍历所有三位数：
for(int i = 100; i < 1000; i++)：这个循环从 100 开始，逐步递增到 999，遍历所有的三位数。
条件判断：
if(pow((i/100),3)+pow((i/10%10),3)+pow((i%10),3)==i)：对于每个三位数 i，分别取出百位数字 i/100、十位数字 i/10%10 和个位数字 i%10。然后计算这三个数字的立方和，如果立方和等于这个三位数本身，则满足条件。
打印满足条件的水仙花数：
printf("%d ", i);：如果条件满足，打印出这个水仙花数。
三、示例
三位数中的水仙花数有：153、370、371、407。当程序运行时，会输出这些数字。
