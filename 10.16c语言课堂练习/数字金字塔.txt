#include<stdio.h>
int main() {
	int n;
	scanf("%d",&n);
	for(int i=1;i<n+1;i++)
	{
		for(int j=1;j<n-i+1;j++)
		{
			printf("  ");
		}
		for(int j=1;j<i+1;j++)
		{
			printf("%d ",j);
		}
		for(int j=i-1;j>=1;j--)
		{
			printf("%d ",j);
		}
		printf("\n");
	}
	return 0;
}
以下是对这段代码的解释：
一、整体功能
这段 C 语言代码的作用是根据输入的整数 n，打印出一个由数字组成的类似三角形的图案。
二、具体步骤
首先读取一个整数 n：
scanf("%d", &n);：从用户输入获取一个整数 n，这个 n 决定了三角形的大小。
外层循环控制行数：
for(int i = 1; i < n + 1; i++)：这个循环从 i = 1 开始，逐步递增到 n，每一次循环代表一行。
第一个内层循环打印空格：
for(int j = 1; j < n - i + 1; j++)：这个循环用于在每一行的开头打印一定数量的空格。随着行数 i 的增加，空格的数量逐渐减少。例如，当 i = 1 时，需要打印 n - 1 个空格；当 i = 2 时，打印 n - 2 个空格，以此类推。
第二个内层循环打印递增的数字：
for(int j = 1; j < i + 1; j++)：这个循环用于打印从 1 开始递增的数字序列。例如，当 i = 1 时，只打印 1；当 i = 2 时，打印 1 2；当 i = 3 时，打印 1 2 3，以此类推。
第三个内层循环打印递减的数字：
for(int j = i - 1; j >= 1; j--)：这个循环用于在递增数字序列后面打印递减的数字序列。例如，当 i = 2 时，在 1 2 后面打印 1；当 i = 3 时，在 1 2 3 后面打印 2 1，以此类推。
最后在每一行的结尾打印一个换行符，使输出形成三角形的形状。
