[实验题5-3]
题目描述
利用键盘随机输入若干个学生成绩(0-100分),然后分别统计0-59,60-79,80-100之间各有多少名学生。写出相应的程序代码。
输入
随机输入若干个学生成绩(0-100分)
输出
输出0-59,60-79,80-100之间各有多少名学生，结果以空格分割
提示
本题一直读到文件末尾
输入文件结束后有自动结束输入的文件结束符。
要想在小黑窗看到结果，数据输入结束后换行再输入ctrl+z模拟文件结束符。
样例输入

0 59 60 100

样例输出

2 1 1

EOF或者1
是scanf("%d",&a)的返回值
#include<stdio.h>
int main() {
	int a,sum1=0,sum2=0,sum3=0;
	while(	scanf("%d",&a)==1) {
		if(a<=59&&a>=0) {
			sum1++;
		} else if(a>=60&&a<=79) {
			sum2++;
		} else if(a>=80&&a<=100) {
			sum3++;
		}
	}
	printf("%d %d %d",sum1,sum2,sum3);
	return 0;
}
以下是对这段代码的解释：
一、整体功能
这段 C 语言代码的作用是统计输入的一系列整数中，处于不同分数区间的数字个数，并输出各个区间的数字个数。具体来说，统计输入中处于 0 到 59 分、60 到 79 分、80 到 100 分这三个区间的数字个数。
二、具体步骤
定义变量：
int a, sum1 = 0, sum2 = 0, sum3 = 0：定义了一个整数变量 a 用于接收输入的数字，以及三个变量 sum1、sum2、sum3 分别用于统计三个不同分数区间的数字个数。
循环读取输入：
while (scanf("%d", &a) == 1)：这是一个循环，只要 scanf 函数成功读取一个整数（即返回值为 1），就会继续循环。在每次循环中，读取一个整数存入变量 a。
分类统计：
根据读取到的整数 a 的值，分别将其归入不同的分数区间进行统计。
如果 a 满足 a <= 59 && a >= 0，说明 a 在 0 到 59 分这个区间，将 sum1 的值加 1。
如果 a 满足 a >= 60 && a <= 79，说明 a 在 60 到 79 分这个区间，将 sum2 的值加 1。
如果 a 满足 a >= 80 && a <= 100，说明 a 在 80 到 100 分这个区间，将 sum3 的值加 1。
输出统计结果：
printf("%d %d %d", sum1, sum2, sum3)：在循环结束后，输出三个分数区间的数字个数。
三、注意事项
这个程序会一直读取输入，直到输入不符合整数格式或者输入流结束。如果希望在特定情况下停止输入，可以在循环中添加适当的条件判断。
代码中没有对输入的合法性进行严格检查，例如输入可能超出整数范围或者不是有效的整数格式。如果需要更健壮的程序，可以添加输入合法性检查的代码。
